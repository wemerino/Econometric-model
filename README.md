# U.S. Public Bus Ridership After COVID-19: Structural Breaks and Recovery Dynamics

## Project Overview

This repository contains datasets, documentation, and analytical resources for the Capstone Project developed as part of the Master of Data Analytics program at the University of Niagara Falls Canada.

The project investigates how COVID-19 altered the determinants of public bus ridership in the United States. Rather than treating the pandemic as a temporary disruption, the study evaluates whether COVID-19 produced structural changes in the relationships between ridership and key economic, policy, and cost drivers.

The analysis is conducted using monthly state-level data to capture differences across U.S. states and to support panel-data modeling approaches.

---

## Objective

The primary objective of this project is to test whether COVID-19 generated structural breaks in the relationship between public bus ridership and its main predictors by comparing pre-COVID and post-COVID periods.

---

## Research Focus

The project evaluates whether the sensitivity of ridership changed after COVID-19 with respect to:

- Economic activity
- Unemployment levels
- Fuel prices
- Restriction intensity or mobility constraints
- Oil shock periods
- Policy environment differences across states
- Workforce composition factors

The analysis focuses on identifying changes in predictor–ridership relationships using regression-based models with interaction terms and fixed effects.

## Intended Outcome

The project aims to provide evidence on whether post-COVID transit demand follows a new behavioral regime, supporting data-driven decision-making for transit planning, policy evaluation, and resource allocation.
